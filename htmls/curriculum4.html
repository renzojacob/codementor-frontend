Understanding JavaScript Closures

A closure is a function that has access to its own scope, the outer function's scope, and the global scope. This allows for powerful programming patterns.

Basic Example

function outerFunction() {
let outerVariable = 'I am outside!';
function innerFunction() {
console.log(outerVariable);
}
return innerFunction;
}

In this example, innerFunction is a closure that captures the outerVariable from its surrounding scope.
