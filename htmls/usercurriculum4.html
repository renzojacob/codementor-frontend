Current Lesson: JavaScript Closures

What are Closures?

A closure is a function that has access to its own scope, the outer function's scope, and the global scope. This allows for powerful programming patterns.

// Example: Basic Closure
function outerFunction() {
let outerVariable = 'I am outside!';
function innerFunction() {
console.log(outerVariable); // Accesses outerVariable
}
return innerFunction;
}

AI Breakdown

Step 1: outerFunction creates a variable. Step 2: innerFunction is defined inside outerFunction. Step 3: innerFunction accesses outerVariable (closure). Step 4: outerFunction returns innerFunction.

Concept Check

What will be logged to the console when this code runs?

function createCounter() {
let count = 0;
return function() {
count++;
return count;
};
}
const counter = createCounter();
console.log(counter());
console.log(counter());

Real-World Usage

Closures are commonly used for:

  • Data encapsulation and privacy
  • Event handlers and callbacks
  • Implementing function factories
  • Module patterns in JavaScript

Common Mistakes

• Creating closures in loops without proper variable scoping
• Not understanding that closures capture variables by reference
• Memory leaks from unnecessary closure retention
